Installing LHAT
===============

Clone the LHAT repository locally

.. codeblock:: python
    git clone https://your_bitbucket_username@bitbucket.org/DeltaresGEO/lsm.git

On command prompt, navigate to the directory and create a conda environment from the yml file.

``conda env create -f environment.yml``
